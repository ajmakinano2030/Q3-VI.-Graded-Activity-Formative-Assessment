# Simple Calculator
This is a basic Python calculator program that performs simple math operations on multiple numbers.

## Features:
- Addition
- Subtraction
- Multiplication
- Division

## How to Use
1. Run the program.
2. Enter numbers separated by commas (example: 10, 5, 2).
3. Choose an operation by entering a number from 1 to 4.
4. The result will be displayed.

## Requirements
- Python 3

## Example Output:

    Enter numbers separated by commas: 8,2

    Choose an operation:
    1 - Addition
    2 - Subtraction
    3 - Multiplication
    4 - Division
    
    Result: 4
